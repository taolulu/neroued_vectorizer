# neroued-vectorizer Web UI 重构方案

> 整理自 2026-04-10 Telegram 对话

## 目标

将 neroued-vectorizer 的 Web UI 从「静态多组件并列布局」重构为**4 状态状态机**，高频操作（上传→矢量化→下载）优先展开，低频操作（预设/参数调整）默认折叠，实现更现代、更紧凑、更有好的用户体验。

---

## 第一组团 — 功能组团设计

### 设计原则

- **高频路径（上传→矢量化→下载）优先显式展开**
- **低频路径（预设/参数调整）默认折叠**
- 同一视觉空间根据当前阶段动态切换内容，无需滚动即可完成核心操作

### 4 状态机定义

| 状态 | 触发条件 | 预览区 | 右上角 | 主体按钮区 |
|------|---------|--------|--------|-----------|
| **空闲态 (idle)** | 页面加载 | 虚线上传框（选择/拖拽/粘贴） | 隐藏 | ✨ 矢量化（禁用） |
| **待执行态 (uploaded)** | 文件上传完成 | 图片预览 | ✕ 清空图标 | ✨ 矢量化（启用） |
| **加载态 (loading)** | 点击矢量化 | 加载动画（覆盖原图） | ✕ 清空图标 | ✨ 矢量化（禁用，loading） |
| **结果态 (result)** | 矢量化完成 | 结果 SVG（原地替换） | ✕ 清空图标 | 💾 下载区 |

### 核心交互设计

1. **预览区作为单一容器**，内部 3 个组件（upload_zone / loading_indicator / result_display）互斥显隐
2. **矢量化按钮 ↔ 下载区同位替换**：两者占同一块视觉空间，切换时无需重新布局
3. **清空按钮小型化**：从独立大按钮改为预览区右上角小图标（✕），上传后出现，结果态保留
4. **gr.Image 原生能力复用**：点击选择、拖拽上传、粘贴上传均由 Gradio 原生支持，无需额外 JS
5. **文件信息简化**：只显示 📐 尺寸 + 💾 文件大小，不显示文件名和色彩模式

### 组件清单

#### 新增 / 修改的组件

| 组件 | 类型 | 用途 | 状态 |
|------|------|------|------|
| `upload_zone` | gr.Image | 上传/预览，idle 和 uploaded 态显示 | 新增 |
| `clear_btn` | gr.Button | 预览区右上角小图标 | 新增 |
| `loading_indicator` | gr.HTML | CSS spinner + 文字加载动画 | 新增 |
| `result_display` | gr.HTML | SVG 结果通过 data URI 渲染 | 新增 |
| `download_area` | gr.Group | SVG/PNG 下载 + 复制源码 | 新增（替换原矢量化按钮位置） |
| `preset_radio` | gr.Radio | 预设选择（照片/插画/线稿） | 从折叠区移出 |

#### 移除的组件

| 组件 | 原因 |
|------|------|
| `comparison_slider` (ImageSlider) | 与状态机设计不符，且占用 400px 空间但信息价值有限 |
| 独立大按钮 `🗑 清空` | 已由右上角图标替代 |
| `📊 结果` 标题区 | 下载区已有 💾 标签，标题冗余 |
| 色彩模式显示 | 文件信息已简化 |

### 预设系统

- **默认值**：「插画」预设
- **外露位置**：第一组团内，折叠区外，常驻显示
- **参数初始化**：页面加载时所有滑块填充为插画预设数值
- **切换行为**：点击即切换，实时更新所有关联滑块

## 第二组团 — 参数配置

- 保留折叠设计（默认 closed）
- 包含：色彩与区域、曲线与形状、高级选项 三组滑块
- 预设选择器已移出，不在此区域

---

## 会话历史

- 保留，但缩小视觉权重
- 位于页面底部
- 点击历史条目可恢复当时的参数设置

## 目标最终布局

```
📤 上传图片
[预览区 — 同一空间，4 状态动态切换]
  idle     → 虚线上传框
  uploaded → 图片预览  [✕]
  loading  → 加载动画  [✕]
  result   → 结果 SVG  [✕]

📐 1920 × 1080  |  💾 2.4 MB

⚡ (●插画) (○照片) (○线稿)

✨ 矢量化                    ← idle/loading 时禁用，uploaded 时启用
  或
💾 导出                      ← result 态时显示
  [下载 SVG]  [下载 PNG]
  [📋 复制 SVG 源码]

⚙️ 参数配置  ▼               ← 折叠区
  🎨 色彩与区域
  📐 曲线与形状
  🔧 高级选项

📜 本次会话历史
```
